## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Running the app


#### Development

- In webpack.config.js set mode to 'development'
```shell
mode: 'development'
```

#### Production

- In webpack.config.js set mode to 'production'
```shell
mode: 'production'
```

#### Run docker

- Go to the backend directory and execute:
```shell
docker-compose up
```
